# 贡献与指点方式

先谢谢你愿意看这个还在成长中的项目。

我们欢迎三类帮助：

1. 直接指出风险：电源、电机、锂电池、接线、机械结构、散热、绝缘，任何安全问题都请优先说。
2. 帮忙纠错：如果文档、BOM、引脚、协议、固件结构里有不合理的地方，请开 issue。
3. 给出更好的路径：如果你知道更省事、更稳、更适合新手复刻的方案，也请告诉我们。

## Issue 建议格式

```text
问题位置：
我看到的风险：
建议改法：
参考资料：
```

## Pull Request 建议

- 保持改动聚焦，一个 PR 尽量只解决一个主题。
- 涉及硬件安全、电源、电机的改动，请说明验证条件。
- 文档尽量使用中文，专业名词可保留英文。
- 固件改动请尽量保证 `firmware/dualeye` 能通过 `idf.py build`。

## 当前最需要的帮助

- Waveshare DualEye 的 ESP-IDF/LVGL 双屏驱动接入。
- DualEye 音频链路和语音交互方案。
- 底盘板固件：DRV8833、限速、超时停车、ACK 协议。
- 铜丝车架的实际装配工艺和强度优化。

我们姿态很低：只要建议具体、有依据，就很宝贵。
