# 机器人积木化设计器第三方组件包扩展规范

版本：2026.06

组件包可以作为单个 JSON 文件导入组件管理页。MVP 阶段导入器支持把旧字段迁移到当前契约，并在导入前展示预检结果。

## 最小组件包

```json
{
  "schema": "robot-components.v1",
  "package": "vendor-sensor-pack",
  "package_label_zh": "厂商传感器包",
  "version": "0.1.0",
  "units": "mm",
  "component_types": [
    {
      "id": "tof_sensor_24x16",
      "category": "sensor",
      "label_zh": "ToF 测距模块",
      "family": "距离传感器",
      "placement": "middle_horizontal",
      "geometry": {
        "kind": "rect_board",
        "length": 24,
        "width": 16,
        "thickness": 4
      },
      "model": {
        "fidelity": "vendor_verified",
        "visual_profile": "rect_board",
        "asset_refs": [
          {
            "kind": "reference",
            "uri": "https://example.com/tof-sensor-datasheet"
          }
        ]
      },
      "ports": [
        { "id": "vcc", "type": "5v" },
        { "id": "signal", "type": "gpio_signal" }
      ],
      "bom": {
        "name": "ToF 测距模块",
        "quantity": 1,
        "required": "optional",
        "search": "ToF sensor module 24x16"
      },
      "sources": [
        "https://example.com/tof-sensor-datasheet"
      ]
    }
  ]
}
```

## 关键约定

- `category` 必须来自契约类别：主控、控制、运动、驱动、电源、传感器、通信、执行器、灯光、音频、紧固/安装、结构。
- `geometry.kind` 决定参数化 3D 渲染方式。当前支持矩形板、盒体、圆柱/轮、齿箱电机、灯环、喇叭、双圆屏板和程序化杆件。
- `model.fidelity` 表示模型可信度：`measured` 实测、`asset_verified` 已由官方 CAD/GLB/STL/STEP 资产校准、`vendor_verified` 厂商规格、`source_estimated` 资料估算、`placeholder` 占位待复核。
- `model.asset_refs` 可引用参考图、GLB/GLTF、STEP/STP、STL。当前先随包展示和导出，后续渲染器可直接加载 GLB，制造侧可引用 STEP/STL。
- 没有 `model` 的旧组件包仍可导入，系统会补为 `source_estimated` 并在预检中提示。
