# Atlas Rover Mk.1 四电机、四轮与 64T 钢齿轮轮方案

## 结论

采用 `N20 + 64T 钢齿轮 3 mm D 孔直接压入`，并改成 4 个电机、4 个轮子，是合理可行的。

但 Mk.1 不建议做“四轮独立控制”。第一版推荐：

```text
4 个 N20 物理电机
  -> 每个 N20 输出轴直接压入 1 个 64T 3 mm D 孔齿轮
  -> 齿轮外圈套 O 型圈/热缩胶圈/橡胶胎提高摩擦
  -> 左前 + 左后视为“左侧逻辑轮”
  -> 右前 + 右后视为“右侧逻辑轮”
  -> XIAO ESP32C3 仍按左右两侧差速控制
  -> 两块 DRV8833 分别驱动前桥和后桥
```

这样能得到接近参考图的四轮姿态，同时固件仍保持 `MOVE/F/B`、`TURN/L/R` 的左右差速模型。不要用一块 DRV8833 长期并联带四个电机；四个 N20 的启动/堵转电流会让小模块发热、掉压或重启。

## 64T 钢齿轮下单前必须确认

| 项目 | 推荐值/要求 | 为什么 |
|---|---|---|
| 数量 | 4 个，建议多买 1-2 个备件 | 四轮方案每个 N20 一个齿轮轮 |
| 齿轮外径 | 30-36 mm 优先 | 接近 34 mm N20 轮胎，底盘高度不用大改 |
| 孔径 | 必须优先 3 mm D 形孔 | 可直接配 N20 3 mm D 轴 |
| 孔型 | D 孔 > 圆孔 > 小孔需扩孔 | D 孔最不容易打滑 |
| 厚度 | 3-6 mm 比较好做轮 | 太薄会晃，太厚太重 |
| 材质 | 钢/黄铜/铝都可，钢最重 | 钢齿轮需要更注意 N20 输出轴侧向负载 |
| 齿形 | 不重要，最好外圈能套胶圈 | 这里不是传动齿轮，是当轮芯用 |
| 螺丝孔 | 有 2-4 个孔最好 | 方便后续改轮毂/夹片 |

如果齿轮没有 3 mm D 孔，也没有螺丝孔，不建议买来做第一版驱动轮。它可以做装饰件，但做驱动轮会很折腾。

## N20 与 64T 齿轮的连接方式

### A. 最推荐：3 mm D 孔齿轮直接压入

适用条件：

- 64T 齿轮中心孔就是 3 mm D 形孔。
- N20 输出轴也是 3 mm D 轴。
- 齿轮厚度不超过 6 mm，装上后不明显晃。
- 四个齿轮来自同一规格，外径一致。

做法：

1. 齿轮对准 D 形轴方向。
2. 用手压入，不要用锤子硬砸电机轴。
3. 如果太松，在 D 平面上贴一小片薄铜箔或 Kapton 胶带微调。
4. 齿轮外圈套橡胶 O 型圈或热缩管，避免钢齿直接打滑、伤桌面。
5. 四个齿轮全部装好后，把电机放在平板上滚动检查，端面摆动尽量小于 1 mm。

风险：

- 齿轮如果只是圆孔，会在 D 轴上打滑。
- 钢齿轮比塑料/橡胶轮重，N20 输出轴承会更吃力。
- 四个刚性轮同时落地时，转向会有侧向摩擦，必须低速调试。

### B. 备选：3 mm D 轴夹紧轮毂 + 齿轮螺丝固定

如果买不到靠谱的 3 mm D 孔齿轮，用轮毂/夹片转接。

```text
N20 3mm D 轴
  -> 3mm D 孔夹紧轮毂/铝轮毂/3D 打印轮毂
  -> 2-4 颗 M2 螺丝
  -> 64T 钢齿轮
  -> 外圈 O 型圈/橡胶圈
```

优点是可拆、可调同心度；缺点是宽度和重量增加，外侧黄铜轮拱要留更多间隙。

### C. 不推荐：胶水硬粘

不建议把 64T 齿轮直接用 502/热熔胶粘在 N20 轴上。

- 很难保证同心。
- 电机一启动容易偏摆。
- 后续无法维护。
- 胶水可能流进 N20 输出轴承。

## 四轮底盘结构

Mk.1 四轮版采用“两侧独立差速”，不是一根通轴。

```text
俯视，车头在上

左前轮/64T  [左前 N20]        18650 / XIAO / DRV8833        [右前 N20]  右前轮/64T
    ◉             |                                           |              ◉

左后轮/64T  [左后 N20]                                      [右后 N20]  右后轮/64T
    ◉             |                                           |              ◉
```

关键尺寸建议：

| 项目 | 建议值 |
|---|---:|
| 底板尺寸 | 110 x 72 mm；如四个电机挤，可放宽到 110 x 78-80 mm |
| 外黄铜框 | 120 x 90 mm |
| 前轮轮心 X 位置 | 距底板前边 18-22 mm |
| 后轮轮心 X 位置 | 距底板后边 18-22 mm |
| 左右轮外宽 | 86-96 mm，按齿轮外径和轮拱实测 |
| 齿轮/轮胎离底板侧边间隙 | >= 2 mm |
| 齿轮离黄铜轮拱间隙 | >= 2 mm |
| 四个轮心高度误差 | 尽量 <= 1 mm |

N20 固定方式：

1. 每个 N20 用一个独立电机夹，或两片 1.5-2 mm FR4/亚克力夹板固定。
2. 四个电机输出轴都朝外。
3. 电机本体贴近底板四角，线从内侧走。
4. 不做贯穿通轴，不把左右轮刚性连起来。
5. 电机固定孔做长槽，允许前后/上下 1-2 mm 微调，保证四轮同时落地。

四轮刚性底盘容易出现一个轮悬空或对角翘起。第一版允许一点点结构柔性：底板用 FR4/亚克力，电机夹不要焊死，先用 M2 螺丝和长槽调平。

## 转向方式

四轮版仍使用左右差速转向：

| 指令 | 左前/左后 | 右前/右后 | 效果 |
|---|---|---|---|
| `MOVE,F` | 前进 | 前进 | 直行前进 |
| `MOVE,B` | 后退 | 后退 | 直行后退 |
| `TURN,L` | 后退 | 前进 | 原地或小半径左转 |
| `TURN,R` | 前进 | 后退 | 原地或小半径右转 |
| `STOP` | 停止 | 停止 | 立即停车 |

注意：四轮都带橡胶外圈时，原地转向需要轮胎打滑，会增加电流和磨损。Mk.1 默认把转向速度限制在 20-35%，持续时间 250-500 ms，先短促转向再微调。

## 两块 DRV8833 接法

推荐前后各一块 DRV8833：

| XIAO ESP32C3 | 接到前 DRV8833 | 接到后 DRV8833 | 作用 |
|---|---|---|---|
| D2 / GPIO4 | AIN1 | AIN1 | 左侧正转 PWM |
| D3 / GPIO5 | AIN2 | AIN2 | 左侧反转 PWM |
| D4 / GPIO6 | BIN1 | BIN1 | 右侧正转 PWM |
| D5 / GPIO7 | BIN2 | BIN2 | 右侧反转 PWM |
| 3V3 | nSLEEP/STBY/EN | nSLEEP/STBY/EN | 若模块有使能脚，拉高 |
| GND | GND | GND | 公共地 |

电机输出：

| DRV8833 | A 通道 | B 通道 |
|---|---|---|
| 前 DRV8833 | 左前 N20 | 右前 N20 |
| 后 DRV8833 | 左后 N20 | 右后 N20 |

也就是说，前后两块 DRV8833 的输入脚并联，输出脚分别接四个电机。固件仍然只知道“左侧”和“右侧”，不需要变成四轮独立 PWM。

不推荐方案：

- 一块 DRV8833 的 A 通道并联左前/左后，B 通道并联右前/右后：能临时空载试转，但不适合作为正式方案。
- 四轮独立控制：需要 8 路 PWM/方向输出和每个轮子的速度补偿，适合 Mk.2，不适合 Mk.1 首版。

## 电源调整

四个 N20 比两个 N20 更吃电。更新后的要求：

| 项目 | 调整 |
|---|---|
| 5 V 升压模块 | 建议 4-5 A 峰值能力，连续电流至少 2-3 A |
| 18650 | 单节高倍率电芯更稳，持续放电建议 >= 8 A；虚标容量电池不要用 |
| 电机电源线 | 20-22 AWG 到驱动板电源母线，24-26 AWG 到单个电机可接受 |
| 电容 | 每块 DRV8833 的 VM/GND 附近并 470-1000 uF + 0.1 uF |
| 开关 | 电机支路独立开关保留，第一次调试必须能一键断电 |
| DualEye | 不给电机供电，只与底盘板共 GND 和 UART |

## 需要额外购买

| 物料 | 数量 | 说明 |
|---|---:|---|
| N20 金属减速电机 | 4 | 6 V，100-150 RPM，3 mm D 形轴，轴长 8-10 mm |
| 64T 钢齿轮 | 4 | 优先 3 mm D 孔，外径 30-36 mm |
| DRV8833 双路 H 桥模块 | 2 | 前桥一块、后桥一块 |
| 橡胶 O 型圈/热缩管 | 4-8 | 套齿轮外圈，提高摩擦 |
| N20 电机夹 | 4 | 比热熔胶可靠 |
| 470-1000 uF 电容 | 2-4 | 每块 DRV8833 至少 1 个 |
| 5 V 大电流升压模块 | 1 | 4-5 A 峰值优先 |

迷你万向轮保留为备选/调试件，不再作为四轮方案的主支撑。

## 推荐验收

1. 四个齿轮装到 N20 上后，手转一圈，端面摆动不超过 1 mm。
2. 四个轮子外径一致，O 型圈厚度一致。
3. 小车放在玻璃板或平整桌面上，四轮都能接触桌面。
4. 车轮悬空发送 `AR1,MOVE,F,25,300`，四轮方向一致。
5. 悬空发送 `AR1,TURN,L,25,250`，左侧两轮后退、右侧两轮前进。
6. 落地低速直行 20-30 cm，不明显跑偏。
7. 左右转都能在 300-500 ms 内产生可见转向，驱动板不过热。

如果跑偏明显，先调机械：四轮同心、左右轮直径、电机固定角度、四轮着地。最后再调固件速度补偿。
