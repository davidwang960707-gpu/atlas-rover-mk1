# Atlas Rover Mk.1 后桥、前桥、底板与 64T 钢齿轮轮方案

## 结论

如果买不到合适的 N20 橡胶轮，可以临时把 `64T 钢齿轮` 做成后轮的刚性轮芯，但不要把它当成“直接套上 N20 就能跑”的成品轮。

Mk.1 推荐的机械路径是：

```text
左右两个 N20 电机独立驱动
  -> 每个 N20 输出轴连接一个轮毂/适配器
  -> 轮毂再夹持或螺丝固定 64T 钢齿轮
  -> 齿轮外圈加 O 型圈/热缩胶圈/橡胶胎提高摩擦
  -> 前方一个万向轮或小脚轮支撑
```

后桥不要做成一根横向通轴。两轮差速需要左右轮能独立转动，否则原地转向和左右转会失效。

## 64T 钢齿轮下单前必须确认

| 项目 | 推荐值/要求 | 为什么 |
|---|---|---|
| 齿轮外径 | 30-36 mm 优先 | 接近原来 34 mm N20 轮胎，底盘高度不用大改 |
| 孔径 | 最好 3 mm D 形孔 | 可直接配 N20 3 mm D 轴 |
| 孔型 | D 孔 > 圆孔 > 小孔需扩孔 | D 孔最不容易打滑 |
| 厚度 | 3-6 mm 比较好做轮 | 太薄会晃，太厚太重 |
| 材质 | 钢/黄铜/铝都可，钢最重 | 钢齿轮需要更注意 N20 输出轴侧向负载 |
| 齿形 | 不重要，最好外圈能套胶圈 | 这里不是传动齿轮，是当轮芯用 |
| 螺丝孔 | 有 2-4 个孔最好 | 方便固定轮毂/夹片 |

如果齿轮没有 3 mm D 孔，也没有螺丝孔，不建议买来做第一版车轮。它可以做装饰件，但做驱动轮会很折腾。

## N20 与 64T 齿轮的连接方式

### A. 最推荐：3 mm D 孔齿轮直接压入

适用条件：

- 64T 齿轮中心孔就是 3 mm D 形孔。
- N20 输出轴也是 3 mm D 轴。
- 齿轮厚度不超过 6 mm，装上后不明显晃。

做法：

1. 齿轮对准 D 形轴方向。
2. 用手压入，不要用锤子硬砸电机轴。
3. 如果太松，在 D 平面上贴一小片薄铜箔或 Kapton 胶带微调。
4. 齿轮外圈套橡胶 O 型圈或热缩管，避免钢齿直接打滑。

风险：

- 齿轮如果只是圆孔，会在 D 轴上打滑。
- 钢齿轮比塑料/橡胶轮重，N20 输出轴承会更吃力。

### B. 推荐：3 mm D 轴夹紧轮毂 + 齿轮螺丝固定

这是最稳的可加工方案。

结构：

```text
N20 3mm D 轴
  -> 3mm D 孔夹紧轮毂/铝轮毂/3D 打印轮毂
  -> 2-4 颗 M2 螺丝
  -> 64T 钢齿轮
  -> 外圈 O 型圈/橡胶圈
```

做法：

1. 选择或打印一个 `3 mm D 孔轮毂`。
2. 轮毂外侧做 2 或 3 个 M2 螺丝孔。
3. 齿轮如果自带孔，按孔距开轮毂孔；如果没有孔，用齿轮夹片夹住，不建议硬钻钢齿轮。
4. 用 M2 x 6 或 M2 x 8 螺丝固定齿轮。
5. 轮毂用顶丝或夹紧结构固定在 N20 D 轴上。

优点：

- 比胶水可靠。
- 齿轮歪了可以拆下来重装。
- 后续换轮胎/齿轮不用换电机。

### C. 可临时用：3D 打印夹片轮

如果没有金属轮毂，可以打印两个夹片，把 64T 齿轮夹在中间。

结构：

```text
外夹片  -> 64T 齿轮 -> 内夹片
       M2 螺丝贯穿夹紧
中心：3 mm D 孔套 N20 输出轴
```

夹片建议参数：

| 项目 | 建议 |
|---|---|
| 外径 | 比齿轮外径小 1-2 mm，不碰齿尖 |
| 厚度 | 单片 2-3 mm |
| 中心孔 | 3 mm D 孔，略紧配 |
| 螺丝 | 3 颗 M2，120 度分布 |
| 材料 | PETG/ABS/尼龙优先，PLA 也可先试 |

### D. 不推荐：胶水硬粘

不建议把 64T 齿轮直接用 502/热熔胶粘在 N20 轴上。

原因：

- 很难保证同心。
- 电机一启动容易偏摆。
- 后续无法维护。
- 胶水可能流进 N20 输出轴承。

## 后桥结构

Mk.1 后桥采用“两侧独立半桥”，不是一根通轴。

```text
左后轮/齿轮   左 N20 电机       中间电池/线束       右 N20 电机   右后轮/齿轮
    |          [固定夹]          18-24 mm 空隙          [固定夹]        |
    |<-- 外侧 -->|                                      |<-- 外侧 -->|
```

关键尺寸建议：

| 项目 | 建议值 |
|---|---:|
| 底板尺寸 | 110 x 72 mm |
| 外黄铜框 | 120 x 90 mm |
| 后轮轮心 X 位置 | 距底板后边 18-22 mm |
| 左右轮外宽 | 86-92 mm |
| 左右 N20 电机间中部空隙 | 18-24 mm |
| 齿轮/轮胎离底板侧边间隙 | >= 2 mm |
| 齿轮离黄铜轮拱间隙 | >= 2 mm |

N20 固定方式：

1. 每个 N20 用一个独立电机夹或两片 1.5-2 mm FR4/亚克力夹板固定。
2. 电机输出轴朝外。
3. 电机本体贴近底板后部，线从内侧走。
4. 两个电机不要共用一根轴，也不要把左右轮刚性连起来。

如果 64T 钢齿轮很重或偏摆明显，后桥增加外侧防摆支撑：

```text
N20 轴 -> 轮毂/齿轮 -> 外侧小支架
```

但第一版不强求外侧轴承。更简单的做法是先减轻轮子、降低速度，并保证齿轮同心。

## 前桥结构

Mk.1 前桥不是驱动桥，只负责支撑和转向随动。

推荐顺序：

1. `10-15 mm 球形万向轮`：最简单，桌面上阻力小。
2. `小脚轮/牛眼轮`：可以，但高度要匹配后轮。
3. `固定前轮轴`：不推荐，会增加转向阻力。

前万向轮位置：

| 项目 | 建议 |
|---|---|
| X 位置 | 距底板前边 12-18 mm |
| Y 位置 | 底板中心线 |
| 高度 | 让底板基本水平，前端可略低 0-1 mm |
| 固定 | M2 螺丝或小夹板，底下可加 1-2 mm 垫片 |

如果后轮换成 64T 齿轮后直径变了，前万向轮必须重新垫高/垫低，让三点同时着地。

## 底板设计 V1.1

底板建议用 1.5-2.0 mm FR4、亚克力、碳板或铝板。先用 2 mm 亚克力/FR4 打样最省心。

### 顶视布局

```text
车头
┌──────────────────────────────┐
│ 前保险杠/灯条预留             │
│          ○ 前万向轮            │
│                              │
│   XIAO/DRV8833/升压模块       │
│                              │
│      18650 电池盒纵向放置      │
│                              │
│ [左 N20]              [右 N20]│
│   ◉ 左后轮/64T        右后轮/64T ◉
└──────────────────────────────┘
车尾
```

### 底板孔位建议

| 区域 | 孔/槽 | 建议 |
|---|---|---|
| 四角铜柱 | M2 圆孔 | 距边 6-8 mm，固定上层黄铜/中层板 |
| 电池盒 | 2 条扎带槽 | 3 x 18 mm，沿车身纵向 |
| N20 电机 | 每侧 2 条长槽 | 2.2 x 12 mm，方便左右微调 |
| 前万向轮 | M2 孔或 2 条小槽 | 允许前后微调 4-6 mm |
| XIAO/DRV8833 | M2 孔或扎带孔 | 模块孔位不统一，先按实物描孔 |
| 线束 | 1.5-2 mm 小孔 | 电机线和电源线用扎带固定 |

### 后轮轴线

先画一条后轮轴线：

```text
距底板后边 20 mm，横向贯穿底板
```

两个 N20 的输出轴中心必须落在这条线上。左、右轴线高度一致，小车才不会一边高一边低。

## 如果坚持用 64T 钢齿轮做轮子

建议额外购买：

| 物料 | 数量 | 说明 |
|---|---:|---|
| 64T 钢齿轮 | 2 | 优先 3 mm D 孔，外径 30-36 mm |
| 3 mm D 孔轮毂/夹紧轮毂 | 2 | 若买不到，就 3D 打印夹片 |
| M2 x 6 / M2 x 8 螺丝 | 6-12 | 固定夹片或轮毂 |
| M2 螺母/防松螺母 | 6-12 | 防松 |
| 橡胶 O 型圈/热缩管 | 2-4 | 套齿轮外圈，提高摩擦 |
| N20 电机夹 | 2 | 比热熔胶可靠 |

## 推荐验收

1. 齿轮装到 N20 上后，手转一圈，端面摆动不超过 1 mm。
2. 左右轮外径一致。
3. 两个后轮同时接触桌面，前万向轮也接触桌面。
4. 悬空发送 `AR1,MOVE,F,25,300`，左右轮方向一致。
5. 落地低速直行 20-30 cm，不明显跑偏。
6. 左右转都能在 300-500 ms 内产生可见转向。

如果跑偏明显，先调机械：轮子同心、左右轮直径、万向轮高度、电机固定角度。最后再调固件速度补偿。
