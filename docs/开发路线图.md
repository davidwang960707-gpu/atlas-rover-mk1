# Atlas Rover Mk.1 开发路线图

这不是承诺书，是一张不断修正的地图。

## Milestone 0：资料整理与方向确认

状态：已完成第一轮。

- 制造图纸包 V1.0。
- 采购清单和规格复审。
- 双板方案确认。
- DualEye UART 接线方案确认。
- VS Code / ESP-IDF / Live Preview 开发环境确认。

## Milestone 1：DualEye HMI 跑起来

状态：进行中。

- 已完成 DualEye 固件 V0.1 骨架。
- 已完成 Web 表情预览。
- 待接入 Waveshare 官方双屏 LCD 驱动。
- 待接入触摸页面切换。
- 待接入音频输入/输出。
- 待把表情参数真正画到两块 240 x 240 圆屏。

验收标准：

- 真机双屏能显示左右眼。
- 语音/触摸事件能切换表情。
- 串口监视器可看到 UART 协议和状态日志。

## Milestone 2：底盘板固件

状态：待开始。

- 选择底盘控制板。
- 实现 DRV8833 电机控制。
- 实现限速、加减速、超时停车。
- 实现 `AR1,ACK,OK/BUSY/ERR`。
- 验证前进、后退、左转、右转、停止。

验收标准：

- DualEye 发送 `AR1,MOVE/F/TURN/STOP` 后，底盘板按预期响应。
- 丢失 UART 指令或异常时自动停车。
- 电机供电不经过 DualEye。

## Milestone 3：机械打样

状态：待打样。

- 铜丝/黄铜车架实际弯折和焊接。
- 底盘、电池、电机、DualEye 的安装位置验证。
- 绝缘、重心、维护空间检查。
- 线束固定和接口防松。

验收标准：

- 车架能承载电池、电路板和双眼头部。
- 电机、轮子、万向轮没有干涉。
- 拆装维护不需要破坏铜丝主体结构。

## Milestone 4：语音与 miniClaw/MimiClaw

状态：接口预留。

- 麦克风收音。
- 唤醒/聆听/思考/说话状态。
- 语义意图转 `atlas_voice_event_t`。
- 移动指令需要确认安全边界。
- TTS 音量驱动 speaking 表情。

验收标准：

- “前进/后退/左转/右转/停止”能稳定映射到底盘指令。
- 非移动类对话不会误触发电机。
- 错误和拒绝场景有明确表情反馈。

## Milestone 5：开源复刻友好度

状态：持续进行。

- 补实物照片。
- 补接线照片。
- 补 BOM 实际购买链接和替代件。
- 补踩坑记录。
- 补最小可运行固件。
- 邀请有经验的朋友 review。

验收标准：

- 新人能按 README 找到入口。
- 复刻者能看懂哪些是稳定方案、哪些还是试验草案。
- Issue 里能沉淀有效建议，而不是散在聊天记录里。
